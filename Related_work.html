Flix Finder Website

This web application is a tool to find any released movie, accompanied by a description, images, and ratings.
The application offers filtering tools to narrow down searches based on type of movie, for example animations, ratings received, and genre of movie.
This application was created with a team of 4, each taking on different tasks to meet the submittal deadline.
The application was designed and styled with HTML, CSS.
The functionality of the website was created using JavaScript.
The movie data displayed was gathered using API calls to OMDB(Online Movie Data Base). We implemented caching as to not make extra calls to the API unless needed, storing previous searches in cache files for the user.



A* Algorithm Visualizer

This visualizer gives a demonstration on how the A* search algorithm traverses a graph to find an endpoint.
The program generates a randomized maze with obstacles, and with a start and end point.
Using the euclidean distance as the heuristic, the program will calculate the best next move to get to the endpoint in the least amount of steps.
This visualizer was created using HTML and p5js for the JavaScript framework. This was a solo project, so no team collaboration was used.
